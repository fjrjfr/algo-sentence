# algo-sentence
This project analyzes a sentence entered by the user and calculates:
the number of words, the length of the sentence and the number of vowels (A,E,I,O,U,a,e,i,o,u)
# variables
char: character 
length-count: INTEGER
words-count: INTEGER
vowels-count: INTEGER
# Example
GomyCode.
length=9
words=1
vowels=3
